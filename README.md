# blinkit-sale-analysis-my-sql-based
SQL project analyzing Blinkit grocery sales data to extract business insights using aggregate functions, filtering, grouping, and ordering queries.
## 🔍 SQL Analysis Performed

Some key analysis performed:

- Total number of products
- Maximum and minimum product price
- Average product weight
- Sales by item type
- Sales by outlet type
- Sales by location tier
- Product visibility analysis
- Sales comparison by fat content

(50+ SQL queries implemented)
